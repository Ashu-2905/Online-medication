Online Medication and Prescription Tracker

Project Overview
The Online Medication and Prescription Tracker is a web-based application that helps users manage their medicines and prescriptions in a simple and organized way. 
It allows users to store prescription details, track daily medications, and get reminders to take medicines on time.
This project is designed as part of a Java Internship to understand full-stack development using modern technologies.

Problem Statement
Many people forget to take medicines on time or lose their prescription records. Managing multiple medicines manually can lead to missed doses, incorrect intake, and health risks. There is a need for a digital system to store prescriptions and track medications easily.

Solution
This application provides a centralized platform where users can:
Register and log in securely
Store and view prescription details
Track daily medications
Receive reminders (future enhancement)
